Server API References 
Below is the list of URLs, parameters, and results to get data from server.

If you want to obtain data, you write JavaScript code as follws:
    h5.ajax({
        type: 'get',
        url: testviewer.utils.formatUrl(*base URL*, *parameters*)
    }).done(this.own(function(result) {
        // Write your proccessing after the data is loaded.
    }));

For example, if you get the detail object whose id is "1", write as follws:
    h5.ajax({
        type: 'get',
        url: testviewer.utils.formatUrl('api/getDetail', {
            id: 1
        }
    }).done(this.own(function(detail) {
        // Write your proccessing after the data is loaded.
    }));

You click a button labeled "Get a sample result" and you can see example data.
